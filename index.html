20KM Brussels

This is a quick & dirty data analysis of the 20KM Brussels' results. Feel free to contact me if you want more information. The source code is available on github.

Who made it ?

78% percent of the participants crossed the finish line. 77% of the women made it to the finish line, 79% for the men.

How fast ?

The average time needed to complete the 20km is 2:00 for both men and women.
For women only it is 2:13, whereas for men only it is 1:55.

Box analysis

As expected, the box 1 is the fastest and the box 6 the slowest. Although not drawn on the graph, the variability in the run time is also increasing with the box number: 16.90 for the box 1, to 23 for the box 6. In a nutshell, being in the box one confirms you run fast, whereas being in the box 6 does not confirm you run slow.

Age analysis

The relationship between the age and the run time is not as strong as I previously expected. It is even start by decreasing for the women. On average, gaining 5 years will make increase your run time by 2%.
